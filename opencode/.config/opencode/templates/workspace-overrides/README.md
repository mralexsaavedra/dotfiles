# Workspace AI overrides (templates)

Use these templates when a specific project needs workspace-level AI instructions.

Design goal:

- Keep OpenCode canonical policy as the base.
- Keep workspace overrides short and project-specific.
- Avoid duplicating global policy blocks.

## Available templates

- `.vscode/opencode-first.instructions.md`
- `.vscode/settings.json`
- `.cursor/rules/opencode-first.mdc`
- `.windsurf/rules/opencode-first.md`
